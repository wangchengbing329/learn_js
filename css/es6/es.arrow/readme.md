##怎么用
省略手法
只有一个参数时，可以省略（）
只有一条语句时 可以省略{} return
##注意
参数之间 不能换行
返回一个对象的时候


##arguments
所有函数里面都可以用的变量,除了箭头函数
参数1[arg1,arg2,arg3]


##区别
- 箭头函数不支持重复命名形参，普通函数可以
箭头函数不能还用call apply等方法改变this
箭头函数没有this，他的this指向定义时所处的上下文（外部函数）的this；
箭头函数没有原型对象，.prototype
- 箭头函数不能作为构造函数
箭头函数没有arguments

- es6 引进的 返回 new 作用的那个构造函数 （构造函数之中）
